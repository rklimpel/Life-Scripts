# Life Scripts

A collection of scripts designed to hack some data of parts of my life. 

## Project Overview

| Project | Status | Description | Tech Stack |
| :--- | :--- | :--- | :--- |
| **[garmin-basketball](./garmin-basketball)** | Working / In Use | Syncs basketball sessions from Google Calendar to Garmin Connect with custom MET-based calorie calculation and EPOC stress adjustment. | Python, Google Calendar API, Garmin API |
| **[garmin-year-comparrison](./garmin-year-comparrison)** | Working | Compares training metrics between two years and generates a side-by-side markdown report with visual charts. | Python, Pandas, Matplotlib, Garmin API |
| **[garmin-year-in-review-md](./garmin-year-in-review-md)** | Working | Generates a detailed annual markdown report and custom charts from Garmin activity data. | Python, Pandas, Matplotlib, Garmin API |

## General Setup

Each script resides in its own subdirectory with a dedicated `README.md`
